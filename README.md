# OSAT Simulator

## 概要

OSAT（Overall Satisfaction Score）を用いて、
現在の満足度から目標達成までの必要回答数をシミュレーションするWebアプリです。

アンケート結果の分析や、
目標OSAT達成に必要な回答数の予測に利用できます。


## 主な機能

- 現在のOSAT計算
- 目標OSAT達成判定
- 必要回答数の計算
- 将来回答の満足率を考慮した予測
- 全回答が「大変満足」だった場合のシミュレーション
- 計算結果の保存
- 履歴管理
- CSV出力
- Excel出力

## 使用方法

1. `index.html` をブラウザで開く
2. 必要な数値を入力する
3. 自動計算された結果を確認する
4. 必要に応じて履歴保存やファイル出力を行う


## 注意事項

本シミュレーターの計算結果は参考値です。
実際の調査分析では、データの品質や回答条件なども考慮してください。
